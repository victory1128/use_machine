📋 使用说明
  1. 顶部填好 「我是 ___」(填一次自动记住),可选拍备注 / 点快捷标签。
  2. 占用卡:默认「按卡占用」模式,点空闲卡(淡绿)即占那一张;切到「全部占用」可点机器「占用」按钮一键占满。
  3. 释放卡:点自己占用的卡(蓝色 ✓)直接释放,或点机器「释放我的」批量释放。
  4. 排队:机器满时点「排队」加入等待队列,有空再来。
淡绿=空闲 淡黄=被占用 淡蓝=我占用 · 顶栏 ⚙ 定时清空 / ⇩ 批量导入 / ✎ 编辑机器

还没有机器,点击右上角「添加机器」开始。
